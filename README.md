# 🐾 Run-cat — 금은동 달려

개발자 집사와 사운드 디자인 집사가  
**우리집 고양이 "금은동"**을 너무 아껴 만든 러닝 게임입니다.  
부족한 실력이지만,  
기획 → 설계 → 디자인 → 개발 → 사운드 디자인까지 **모든 과정을 직접 진행**했습니다.

---

## 🎮 게임 개요
- **장르:** 러닝 액션 게임
- **주인공:** 금은동(우리집 고양이)
- **제작 배경:** 유튜브 콘텐츠 기반 제작
- **그래픽:** 일부 무료 에셋 + 직접 디자인 제작

---
