> 개발자 집사 🖥 + 사운드 디자인 집사 🎧가 함께 제작했습니다.

## 🎮 게임 개요
| 항목 | 내용 |
|------|------|
| **장르** | 러닝 액션 게임 |
| **주인공** | 금은동 (집사들의 고양이) |
| **제작 배경** | 유튜브 콘텐츠 기반 제작 |
| **그래픽** | 일부 무료 에셋 + 나머지 직접 제작 |
