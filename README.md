# Run-cat
금은동 달려
개발자 집사와 사운드 디자인 집사가 고양이를 너무 아껴서 만든 우리집 고양이 "금은동"을 위한 게임

부족한 실력이지만 
기획,설계,디자인,개발,사운드 디자인까지 포함하여 만들었음

해당 게임은 유튜브를 토대로 만들었으며 
사용된 asset 중 몇개는 무료 aseet 을 사용하였고 다른 부분들은 직접 디자인을 하였음
